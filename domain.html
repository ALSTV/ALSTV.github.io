Warum diese Komische Domain?

Der Server ist für uns ein leidenschafts Projekt, er bringt uns kein Geld ein.
Also wollen wir auch möglichst wenig geld ausgeben. Deshalb verzichten wir auf einen Server Hoster und Hosten den Server zu Hause. Ich will Portfreigaben vermeiden, da sonst Hacker einfacher in mein Netzwerk kommen. Deswegen setzten wir auf Tunneling Dienste. Das bedeutet, das man sich nicht über den Router mit dem Server verbindet, sondern über einen Server des Tunneling Dienst auf unseren Server kommt. Das ist viel Sicherer. Wir verwenden den Dienst playit.gg . Dort kann man sich leider nicht selbst eine Domain aussuchen aber ich habe mich bemüht die schönste zu bekommen.
